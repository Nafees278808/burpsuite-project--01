# BurpSuite Project --01

This repository contains BurpSuite work files, configurations, and screenshots used for web application security testing.

## 📂 Files Included
- `bruipsuite.json` – BurpSuite configuration
- `proxy.json` – Proxy settings
- `dvwa from metasploit` – DVWA testing notes
- `exact paasword` – Password testing data (excluded via `.gitignore` for safety)
- `README.md` – Project documentation
- `.gitignore` – Exclusion rules for sensitive files
- Screenshots (`.png`) – Captured workflow steps
