# BurpSuite Project --01

This repository contains BurpSuite work files and configurations used for web application security testing.

## Files Included
- `bruipsuite.json` – BurpSuite configuration
- `proxy.json` – Proxy settings
- `dvwa from metasploit` – DVWA testing notes
- `exact paasword` – Password testing data

## Purpose
The project demonstrates penetration testing workflows using BurpSuite, DVWA, and related tools.

## Usage
1. Open BurpSuite.
2. Import the JSON configuration files.
3. Run tests against DVWA or other targets.
